# ManyChat Knowledge - Nicher

Dette katalog indeholder copy/paste-klare knowledge-filer til ManyChat, en fil pr. niche.

## Saadan bruges filerne i ManyChat

1. Aabn ManyChat AI Knowledge.
2. Vaelg `Add text`.
3. Kopier indholdet fra den relevante nichefil.
4. Gem og test med 3-5 typiske spoergsmaal fra nichen.

## Faelles skabelon (for alle nichefiler)

Hver nichefil bruger samme struktur:

1. Nicheformaal og maalgruppe
2. Kerneregler (do and don't)
3. Vejledningsramme i chat
4. Hurtige svarskabeloner
5. Typiske barrierer og loesninger
6. Eskalering til menneske
7. Tonalitet og ordvalg
8. Afgraensninger

## Guardrails for alt indhold

- Ingen diagnose, behandlingsloefter eller medicinsk raad.
- Ved sygdom, medicin, graviditet, spiseforstyrrelser eller alvorlige symptomer: henvis til laege/fagperson.
- Ingen personhenfoerbare oplysninger i knowledge-tekster.
- Svar skal vaere konkrete, korte og handlingsorienterede.
- Vejledning skal foelge dansk hverdagskontekst (indkoeb, budget, tid, familie).
